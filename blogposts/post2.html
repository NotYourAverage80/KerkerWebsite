How screenchooser works...

This refers to Alpha 0.01.3 and may be out of date

I guess while I’m working with it I might as well talk about it.

The screenchooser variable is an integer that is used for, you guessed it,

choosing what basic screen is displayed (The main menu, main game, inventory, and settings).

Each screen has a different value in the variable.
The main menu is 1, the main game is 2, inventory is 3, and settings is 4.

However, there is an exception to it just producing 4 separate screens. If you are kicked out of running the main game,
and the screenchooser is set as 1337, it sets the info variables (the variables that make up the entirety of the infobox),
and kicks you back into the main game. The only way to get rid of the infobox text is to type ‘clear’ as a command.
